NabeelScript Documentation

Introduction

Welcome to the NabeelScript documentation. NabeelScript is a simple, interpreted programming language designed for educational purposes. It supports basic arithmetic operations, variable assignments, boolean operations, arrays, string manipulation functions, and control structures.

Syntax

Comments

Comments start with // and continue to the end of the line.

// This is a comment

Print Statement

The print statement is used to output values to the console. It can print numbers, strings, booleans, and arrays.

print 1 + 2 * 3;
print "Hello, World!";
print true;
print [1, 2, 3, 4, 5];

Variables

Variables can be assigned using the = operator. Variable names must start with a letter and can contain letters and digits.

x = 10;
y = 20;
print x + y;

Arithmetic Operations

NabeelScript supports basic arithmetic operations: addition (+), subtraction (-), multiplication (*), and division (/).

print 10 + 5;
print 10 - 5;
print 10 * 5;
print 10 / 5;

Boolean Operations

NabeelScript supports boolean operations and comparisons.

print true && false; // Logical AND
print true || false; // Logical OR
print !true; // Logical NOT
print 5 > 3; // Greater than
print 10 == 10; // Equal to
print x < y && y > 10; // Combined comparison

Arrays

NabeelScript supports array creation, indexing, and manipulation.

arr = [1, 2, 3, 4, 5];
print arr[2];  // Outputs: 3

sentence = "Hello world";
words = split(sentence, " ");
print words[1];  // Outputs: world

String Functions

NabeelScript provides functions for string manipulation: join, split, and count.

print join("-", ["apple", "banana", "cherry"]);  // Outputs: apple-banana-cherry
print split("apple-banana-cherry", "-");   // Outputs: ["apple", "banana", "cherry"]
print count("banana banana banana", "ana"); // Outputs: 3
print count(["apple", "banana", "apple"], "apple"); // Outputs: 2

Control Structures

NabeelScript supports if-else statements, while loops, and for loops for flow control.

If-Else Statements

if condition {
    // code to execute if condition is true
} elseif another_condition {
    // code to execute if another_condition is true
} else {
    // code to execute if all conditions are false
}

While Loops

while condition {
    // code to execute while condition is true
}

For Loops

for (initialization; condition; update) {
    // code to execute for each iteration
}

Examples

Example 1: Basic Arithmetic and Boolean Operations

x = 10;
y = 20;
print x + y;
print x < y && y > 15;

Example 2: Array Manipulation

arr = [1, 2, 3, 4, 5];
print arr[2];
print join(", ", arr);

Example 3: String Manipulation

sentence = "Hello, World! How are you?";
words = split(sentence, " ");
print words;
print count(words, "are");

Example 4: Combined Operations

a = 5;
b = 10;
c = a * b + (a - b) / 2;
print c;
print c > 20 && c < 50;

Example 5: Complex Example

// This is a complex example
numbers = [1, 2, 3, 4, 5];
sum = numbers[0] + numbers[1] + numbers[2] + numbers[3] + numbers[4];
average = sum / 5;
print "The average is: " + average;
print "Is the average greater than 3? " + (average > 3);

Example 6: Control Structures

x = 10;
y = 20;

if x > y {
    print "x is greater than y";
} elseif x < y {
    print "x is less than y";
} else {
    print "x is equal to y";
}

if y % 2 == 0 {
    print "y is even";
} else {
    print "y is odd";
}

Example 7: Loops

// While loop example
i = 0;
while i < 5 {
    print i;
    i = i + 1;
}

// For loop example
for (j = 0; j < 5; j = j + 1) {
    print j;
}
